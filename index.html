Kitab Gaul Update 10/12/2015

Exciting news! Last week we deployed a RESTful API to access Kitab Gaul data. Today, we published the public RESTful API documentation. Developers now have access to search, entries, and member information. Limited read-only access is available today, write access will be available in the future.

You can read the API documentation at https://kitabgaul.com/api.

We’ve added a link to the weekly updates and general stories to the footer of every page, and we’ve fixed minor bugs.

We’re busy working on Android and iOS apps, we’ll let you know when we have them complete!

Kitab Gaul Update 11/9/2015

We have an app!

Kitab Gaul now has an app for Android. You can get free it via Amazon App Store. This initial release allows you to search and read entries in Kitab Gaul. It does not allow you to modify anything yet, we’re still working on it. iOS app is also still being worked on.

Please try the app out and if you have any questions/comments, please send us a message.
